Job Traveler

This workorder closing sheet was created in Crystal Reports and was designed to provide costing and operational information upon completion of production. It is still being used in the shop it was created for.

The added value to this was the barcode across the top. This has a file-name friendly, keyword searchable name that software attached to the copy machine scanner, would read and use as an indexible file name.
