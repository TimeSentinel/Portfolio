Case Study: Flying Sandwich
IBM Full Stack Developer Program: Javascript Essentials
Interactive Site

This project was for my IBM certificate. The focus is on AJAXj, APIs and JSON feeds/files.

Needed to be a three page site: Home, About, and Contact.

Of course, the business itself, is fictional.

Menu/Nav Bar

This needed to be fully functional, CSS styled, and responsive. Every page was designed to fit and look well on every size device screen.

API, JSON

The primary focus: retrieving remote data

In this example, I created a JSON file with the travel data. By using the search field in the nav bar, you could retrieve the specific data you were looking for.

I really like projects involving external data; databases, external files, APIs, etc.

To me, data storage is one of the most useful technologies I can use in web development.
